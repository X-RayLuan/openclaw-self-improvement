# v0.2.11 — Agent repair loop from scorecards

Turns agent scorecards into an actionable repair loop.

Changes:
- Adds repair classification to failed, blocked, and pending scorecard lanes.
- Adds deterministic recovery ticket generation with `daily-agent-scorecard.mjs --repair`.
- Adds repeat-count detection for `agent + lane + failureClass` patterns.
- Marks repeated failures as `EXPERIMENT_REQUIRED` so they become self-improvement experiments instead of blind retries.
- Adds tests for repair action classification and ticket writing.

Verification:
- `node --test tests/harness-repair-loop.test.mjs`
- `node --check scripts/harness-improvement-lib.mjs`
- `node --check scripts/daily-agent-scorecard.mjs`
- `node scripts/daily-agent-scorecard.mjs --repair --tickets-dir /tmp/openclaw-self-improvement-repair-test --json`
