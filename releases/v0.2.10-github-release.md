# v0.2.10 — Harness observability and daily scorecards

Adds OpenClaw harness observability helpers for classifying repeated runtime failures and generating daily agent delivery scorecards.

Changes:
- Added harness failure taxonomy for gateway, provider, session, skill, tool, platform, and approval blockers.
- Added `scripts/analyze-openclaw-failures.mjs`.
- Added `scripts/daily-agent-scorecard.mjs`.
- Added bin aliases `openclaw-analyze-failures` and `openclaw-agent-scorecard`.
- Updated SKILL.md so repeated agent delivery failures can become logged errors, experiments, or promoted guardrails.

Verification:
- `node --check scripts/harness-improvement-lib.mjs`
- `node --check scripts/analyze-openclaw-failures.mjs`
- `node --check scripts/daily-agent-scorecard.mjs`
- `node scripts/daily-agent-scorecard.mjs --json`
